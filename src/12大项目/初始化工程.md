1. npm init

2. npm i typescript -g

3. tsc --init  创建ts配置文件

4. npm install -D tslint typescript
5. npx tslint --init

# 这里装一下egg
# 发现他都给你弄好了,不用上面的东西了
1. npm i egg-init -g
2. npm init egg --type=ts

egg-bin 已经内建了 ts-node ，egg loader 在开发期会自动加载 *.ts 并内存编译
